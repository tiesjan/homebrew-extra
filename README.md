# Homebrew Extra

Extra formulae for the Homebrew package manager.

## Available formulae
- cd-discid

## How do I install these formulae?

`brew install tiesjan/extra/<formula>`

Or `brew tap tiesjan/extra` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "tiesjan/extra"
brew "<formula>"
```
